# 📊 Task 2 - Exploratory Data Analysis (EDA) & Business Intelligence

## 📌 Project Overview

This project was completed as part of the **ApexPlanet Data Analytics Internship**.

The objective was to perform Exploratory Data Analysis (EDA) on a sales dataset, answer business questions using SQL, and design a static dashboard to present key insights.

---

## 🎯 Objectives

- Analyze sales data using Python
- Perform descriptive statistical analysis
- Generate meaningful visualizations
- Execute SQL business queries
- Design a static dashboard in Microsoft Excel

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQLite
- Microsoft Excel
- Git & GitHub
